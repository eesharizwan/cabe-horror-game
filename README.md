# CABE Horror Game

A collaborative 2D horror-survival game developed in Godot for CS 292.

The game places the player inside a dark and hostile environment where they must survive, explore, manage limited resources, and escape while avoiding threats throughout the map.

Built using the Godot game engine and GDScript, the project focuses on gameplay systems, atmosphere, player interaction, and horror mechanics.

# How To Run The Project

## Step 1 — Install Godot

Download Godot:

https://godotengine.org/download

---

## Step 2 — Download or Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/cabe-horror-game.git
```

Or download the ZIP directly from GitHub.

---

## Step 3 — Open the Project

1. Open Godot
2. Click "Import"
3. Select the project folder
4. Open the project

---

## Step 4 — Run the Game

Press the play button inside Godot or press:

```txt
F5
```
